# BMI Application

> A beatifully designed application with which users can use to calculate their BMI and Ponderal indexes to help them keep track of their health.

## Built With
- Kotlin

### Dependencies Used
- Fragment & Fragment Navigation
- Splashscreen API
- Google AdMob
- Truth
- Espresso
- Mockito
- Fragment Testing
- TestNavController
